## Quick Start

1. Open `index.html` in a modern browser
2. Start typing markdown in the editor
3. See live preview on the right

## Keyboard Shortcuts

| Shortcut | Action          |
|----------|-----------------|
| Ctrl+B   | Bold            |
| Ctrl+I   | Italic          |
| Ctrl+K   | Insert Link     |
| Ctrl+S   | Save to Browser |
| Ctrl+O   | Open File       |
| Ctrl+N   | New Document    |
| Ctrl+F   | Find & Replace  |
| Ctrl+1   | Raw View        |
| Ctrl+2   | Split View      |
| Ctrl+3   | Preview View    |

## View Modes

- **Raw** - Full editor, markdown only
- **Split** - Side-by-side editor and preview
- **Preview** - Rendered markdown only

## Technologies

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- [marked.js](https://marked.js.org/) - Markdown parsing
- [highlight.js](https://highlightjs.org/) - Syntax highlighting
- [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/) - PDF export
